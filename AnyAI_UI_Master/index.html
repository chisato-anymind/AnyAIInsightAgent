コンポーネントを見る パターンとガイド テンプレート

1. トークン駆動

色・タイポ・スペーシング等を CSS 変数に集約。ダークモードやブランド調整が 1 箇所で完結。

2. パターン指向

一覧/詳細・作成フォーム・検索/フィルタ・空/ローディング/エラーなど、実務の流れに直結。

3. ビルドレス

HTML + CSS + JS だけ。React/Vue でも、サーバーサイドレンダリングでも流用しやすい構造。

共通要件の自動化

  • 全ページ左上に assets/AnyAI_logo.png を挿入（適切な高さ 28px）。
  • ファビコンに assets/AnyAI_icon.png を採用。
  • アクセシビリティ: フォーカスリング / コントラスト / キーボード操作に配慮。
